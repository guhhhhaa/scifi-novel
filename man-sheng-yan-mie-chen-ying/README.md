## 简介

因为无节制的求生欲望和连续失误，人类创造出的“蔓生物”慢慢将他们送入地狱。

……纳米战争，深渊虫洞，巴别塔，戴森球……

人类文明的命运，被两位少年陈颖和谷妄蓝于无意之中改写。
